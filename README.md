# AVR-Processor
VHDL implementation of an 8-bit AVR processor supporting 29 whole instructions!

## Project Information
Caltech EE 119B
Albert Gural and Bryan He

## Instructions supported
* ADC
* ADD
* ADIW
* AND
* ANDI
* ASR
* BCLR
* BLD
* BSET
* BST
* COM
* CP
* CPC
* CPI
* DEC
* EOR
* INC
* LSR
* MUL
* NEG
* OR
* ORI
* ROR
* SBC
* SBCI
* SBIW
* SUB
* SUBI
* SWAP
